Q3. Internet Connectivity Check

1. Command to check internet access:
   ping -n 1 www.google.com (Windows)

2. Command to check if a website is reachable:
   ping www.google.com

3. Command to show network configuration:
   ipconfig

4. How to test public IP:
   curl ifconfig.me or open https://whatismyipaddress.com

5. Continuous ping command:
   ping -t www.google.com
